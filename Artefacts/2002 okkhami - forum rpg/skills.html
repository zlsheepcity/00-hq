Рукопашный бой

До того как боги принесли на землю оружие, каждое существо использовало в боях свои физические особенности. Кто-то дрался рогами, кто-то острыми когтями и зубами, кто-то кулаками. Уже к тому времени появились первые зачатки «рукопашного боя» как профессионального умения.
• Формула атаки:
атака = ( Сила * 2 + Ловкость * 2 ) * ( значение_навыка * 100% )
• Бонус к защите против рукопашного противника:
защита = ( Сила * 2 + Ловкость * 2 ) * ( значение_навыка * 100% )
• Против вооружённого противника бонуса к защите не даёт.
Изначально навык доступен всем.
Воин: 20%
Вор: 15%
Маг: 10%
Маг-Воин: 25%

Навык кинжала
• Формула атаки:
атака = ( Ловкость * 6 ) * ( значение_навыка * 100% )
• Бонусов к защите не даёт
Вор обладает этим навыком изначально ( 10% ).
Маг-Воин не может иметь значение навыка превышающее 50
Все классы одинаково равномерно изучают этот навык

Навык тупого оружия
• Формула атаки:
атака = ( Сила * 2 + Воля * 6 ) * ( значение_навыка * 100% )
• Бонус к защите:
защита = ( Сила + Воля * 4 ) * ( значение_навыка * 100% )
Маг-Воин обладает этим навыком изначально ( 10% ).

Навык меча
Меч - оужие созданное для войны. Созданное воинами для воинов по праву считаеться самым грозным оружием. От лёгких эльфийских до тяжёлых двуручных меч в умелых руках является в первую очередь признаком воинской касты.
• Формула атаки:
атака = ( Сила * 6 + Ловкость * 3 ) * ( значение_навыка * 100% )
• Бонус к защите:
защита = ( Сила * 4 + Ловкость * 2 ) * ( значение_навыка * 100% )
Вор и Маг с большим трудом способны изучить этот навык.

Навык топора
• Формула атаки:
атака = ( Сила * 10 ) * ( значение_навыка * 100% )
• Бонус к защите:
защита = ( Сила ) * ( значение_навыка * 100% )
Вор и Маг с большим трудом способны изучить этот навык.

Навык копий
• Формула атаки:
атака = ( Сила * 3 + Ловкость * 5 ) * ( значение_навыка * 100% )
• Бонус к защите:
защита = ( Сила * 3 + Ловкость * 5 ) * ( значение_навыка * 100% )
Маг с большим трудом способен изучить этот навык.

Парирование
Парирование можно совершить с оружием в руках против вооружённого же противника. Суть его заключаеться в том, что ловким движение мокружие противника отводиться в сторону от удара.
Когда атака противника пропущена, возникает парирование, способное с вероятностью ( значение_навыка * 100% ) нейтрализовать удар. Действует только против оружия.
Воину навык парирования доступен изначально ( 5% ).

Ярость
Ярость воина возникает во время боя и позволяет нанести противнику более точный удар.
Перед каждой атакой с вероятностью ( значение_навыка * 100% ) возникает ярость. При возникновении, ярость увеличивает атаку воина на 50%. Действует только при атаке оружием или рукопашной.

Крик
Воин издаёт жуткий крик, понижающий защиту всех противников вдвое.
Используется вместо атаки. С вероятностью ( значение_навыка * 100% ) защита всех противников снижаеться на 50% на этот и следующий туры.

Контратака
Отбившись от атаки воин способен сделать выпад вне очереди.
Когда вражеская атака отбита. С вероятностью ( значение_навыка * 100% ) воин может сделать ответную атаку вне очереди.

Разоружение

Позволяет выбить оружие из рук противника, после чего тот вынужден некоторое время драться без оружия.
Используеться вместо атаки. Вместо урона противник теряет оружие на текущий и следующий тур.
Все формулы атаки беруться как и при обычной атакой оружием.
Битовый навык (либо есть, либо нету)
Вору доступен изначально.

Уклонение
Игрок пытается уклониться в течении двух туров.
Используеться вместо атаки. На два тура к защите прибавляеться бонус.
бонус = (Ловкость * 3 + Внимательность * 2 ) * ( значение_навыка * 100% ).

Удар в спину
При удачном ударе в спину, вор наносит тройной урон противнику.
Используеться вместо атаки. С вероятностью ( значение_навыка * 100% ) вор подбираеться к противнику и наносит тройной урон.
